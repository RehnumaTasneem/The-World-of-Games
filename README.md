# The World of Game

## Description
**The World of Game** is a collection of classic and popular games implemented in a single project. It includes **Tetris, Candy Crush, and Tic-Tac-Toe**. This project demonstrates game logic, graphics handling, and user interaction.

## Games Included

### Tetris
- Falling block mechanics
- Line clearing and scoring
- Keyboard controls

### Candy Crush
- Match 3 or more candies
- Score tracking
- Level progression

### Tic-Tac-Toe
- 3x3 grid gameplay
- Player vs Player mode
- Win, Draw, and Restart functionality

## Tools & Technologies
- Programming Language: [C++]  
- Graphics/Engine: [OpenGL/GLUT]  

## How to Run
1. Clone the repository.  
2. Compile and run the project using the appropriate compiler or IDE.  
3. Select a game to play from the main menu.
